# Python Bank System 

A simple bank system made with Python.

This project allows you to manage bank accounts, including deposits, withdrawals, transfers, and transaction history.  
All data is saved automatically using a JSON database file.

---

## Features

- Create bank accounts
- View all accounts
- Deposit money
- Withdraw money
- Transfer money between accounts
- View transaction history
- Delete accounts
- Data persistence using `.json` file

---

## Requirements

- Python 3.x

---



## Installation / Clone /

```bash
git clone https://github.com/Huntervelasco/Bank-Management
cd Bank-Management
python3 code.py

```
